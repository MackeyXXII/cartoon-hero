Definitions:

There are two main programming languages that are being used in this webpage:

HTML


We use HTML (Hypertext Markup Language) in a webiste to declare the behavior of the website. We tell it what elements it should display, how they should be orderd and where should it search from them.

CSS


CSS is the language we use to style an HTML document. CSS describes how HTML elements should be displayed. It is mainly concerned for the formatting of the elements of the webpage. CSS needs HTML elements to work.
